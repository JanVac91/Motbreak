## üöÄ GETTING STARTED

### Installation
1. Open Blender (3.0 or newer)
2. Go to `Edit ‚Üí Preferences ‚Üí Add-ons ‚Üí Install`
3. Install both files:
   - `mot_importer_v1_8.py` (for importing)
   - `mot_exporter_v2_8.py` (for exporting)
4. Enable both add-ons by checking their boxes

## üìê SKELETON SETUP

### Required Structure
Your character skeleton must have bones named exactly:
```
Node0
Node1
Node2
Node3
Node4
... (up to Node21 for human characters)
```

**Important:**
- Node0 must be the armature (the main object)
- Node1-21 must be bones inside the armature
- Names are case-sensitive!

### Project Settings
Before animating, set these in Blender:
1. **Frame Rate:** Go to `Output Properties ‚Üí Frame Rate ‚Üí 60`
2. **Rotation Mode:** Select all bones ‚Üí Press `R` ‚Üí Choose `XYZ Euler`

---

## üé® ANIMATING

### Basic Rules
1. **Root Motion (walking/running):**
   - Put forward/backward movement on **Node0** (Y location)
   - Put left/right movement on **Node0** (X location)

2. **Body Rotations:**
   - Animate **Node2-21** normally (head, arms, legs, etc.)
   - Use rotation keyframes

3. **Interpolation:**
   - For location (movement): Set to **Linear** interpolation
     - Select keyframes in Graph Editor
     - Press `T` ‚Üí `Linear`
   - For rotation: Use default (Bezier is fine)

### Timeline
- Set your animation range in the Timeline
  - Example: `Start Frame: 0`, `End Frame: 120`
- The exporter will only export keyframes within this range

## üì• IMPORTING ANIMATIONS

### Basic Import
1. `File ‚Üí Import ‚Üí Outbreak Import (.mot)`
2. Select your `.mot` file
3. Click **Import**

This will replace any existing animation.

### Import Options

**‚òê New Action**
- Check this to keep your current animation
- Creates a separate animation you can switch between
- **When to use:** You want to import multiple animations (idle, walk, run, etc.)

**‚òê Append Mode**
- Check this to add animation after your current animation
- Move the timeline cursor to where you want it to start
- **When to use:** You want to chain animations together (walk ‚Üí run)

**‚òê Ignore Face (0x06)**
- Check this to skip facial animations
- **When to use:** Your character doesn't have facial bones

**‚òë Monster: First Section Only**
- Leave this checked (default)
- Prevents glitchy animations on monsters
- **When to use:** Always for monster animations

---

## üì§ EXPORTING ANIMATIONS

### Basic Export
1. Set your frame range in the Timeline
   - `Start Frame: 0`
   - `End Frame: 120` (or your animation length)
2. `File ‚Üí Export ‚Üí Capcom Outbreak (.mot)`
3. Choose where to save
4. Click **Export**

### Export Options

**‚òê Enable Loop**
- Check this for animations that repeat (walk, run, idle)
- Leave unchecked for one-time animations (pick up object, open door)

**Loop Frame**
- Usually set to `0`
- This is the frame where the animation loops back to

## ‚ö†Ô∏è COMMON PROBLEMS

### "Character slides on the ground"
**Fix:**
1. Make sure Node1 has the Z location animation
2. Set location keyframes to Linear interpolation:
   - Graph Editor ‚Üí Select location curves
   - Press `T` ‚Üí Linear

---

## üìã QUICK CHECKLIST

Before exporting:
- [ ] Frame rate is 60 FPS
- [ ] Rotation mode is XYZ Euler (all bones)
- [ ] Timeline range is correct (Start/End frames)
- [ ] Node1 has forward movement, Node2 the offset movement
- [ ] Location keyframes are Linear interpolation
- [ ] Loop setting is correct (on for walk/run, off for actions)

---

## üéì TIPS

1. **Test early!** Export and test in-game frequently, don't wait until the animation is perfect
2. **Name your actions** - Makes it easier to find animations later
3. **Use Linear for movement** - Bezier curves cause sliding
4. **Keep it simple** - Start with simple animations before complex ones
5. **Save often** - Blender can crash, save your work!
